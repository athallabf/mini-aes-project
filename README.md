## Onboarding

1. pastikan `uv` sudah terinstall https://docs.astral.sh/uv/getting-started/installation/
2. `uv python install 3.10`
3. `uv venv` (pastikan menggunakan python >=3.10 )
4. `source .venv/bin/activate`
5. `uv run main.py`
